# SmartCampus – Stevens IT Quackathon 2025 

**SmartCampus** is a Flutter-based mobile application designed to improve the student experience at Stevens Institute of Technology.

We tackled three core problems students face:
- Getting lost navigating campus
- Shuttle booking confusion during peak hours
- Emergency response delays

---

## Key Features

- **AI-Backed 3D Navigation** – Indoor & outdoor wayfinding
- **Smart Shuttle Booking** – Queue tracking + live shuttle data
- **Emergency Contact Access** – One-tap link to campus security and health
- **Next Steps** – AR view, AI alerts, multilingual support

---

## 🛠️ Tech Stack

- **Frontend**: Flutter (Dart)
- **Design**: Figma
- **Backend**: Prototype using Python APIs (future integration with IoT + Azure)
- **AI & Cloud**: Microsoft Azure OpenAI (conceptual)
- **Mapping**: 3D Map APIs + GPS tracking
